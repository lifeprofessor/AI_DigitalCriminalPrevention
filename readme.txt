1. **도입부 안내 문구**

2. **디지털 범죄의 유형과 위험 이해하기**
   - 보이스피싱, 스미싱, 메신저피싱 등의 정의
   - 각 범죄 유형별 실제 사례 설명
   - 특히 고령층을 노린 최신 수법과 피해 사례 소개

3. **딥페이크의 원리와 구별법 익히기**
   - 딥페이크가 어떻게 만들어지는지 간단히 설명 (텍스트, 이미지, 음성, 영상 포함)
   - 사회적 위험성과 사례 제시
   - 진짜와 가짜를 구분하는 실전 팁 제공

4. **디지털 범죄 대응 전략과 대화법**
   - 상황별로 사용할 수 있는 대응 요령 (예: 수상한 전화/문자 수신 시)
   - 신고 방법 안내
   - 시니어를 위한 쉽고 명확한 대응 문구 예시 제시

5. **디지털 범죄 예방 실천 방법**
   - 출처 불명 링크 클릭 금지, 통신사 차단 기능 설정 등 실천 가능한 수칙 정리
   - SNS, 메신저 사용 시 개인 정보 보호법 및 주의사항 정리

6. **마무리 섹션**
   - 학습 내용 요약
